# 如何新增部落格文章

## 📝 快速流程

### **方式 1：直接編輯程式碼（最簡單）**

1. **打開檔案**
   ```
   /app/blog/page.tsx
   ```

2. **在 `blogPosts` 陣列加入新文章**
   ```typescript
   const blogPosts: BlogPost[] = [
     // 現有文章...
     {
       id: '2',  // 遞增編號
       title: '你的文章標題',
       description: '文章簡介（1-2 句話）',
       date: '2026-01-30',  // 發布日期
       category: '補品營養',  // 或 '血檢數據'、'訓練恢復'、'個案追蹤'
       readTime: '5 分鐘',
       slug: 'your-article-slug'  // 英文網址（不要有空格）
     },
   ]
   ```

3. **建立文章內容**
   - 打開 `/app/blog/[slug]/page.tsx`
   - 在 `blogContent` 物件加入內容：
   
   ```typescript
   const blogContent: Record<string, {...}> = {
     // 現有文章...
     'your-article-slug': {
       title: '你的文章標題',
       date: '2026-01-30',
       category: '補品營養',
       readTime: '5 分鐘',
       content: `
   ## 你的第一個標題
   
   這是段落內容。**粗體文字**可以這樣寫。
   
   ### 子標題
   
   - 列表項目 1
   - 列表項目 2
   
   ---
   
   ## 免責聲明
   
   **這是我的個人實驗紀錄，不構成醫療建議。**
       `
     }
   }
   ```

4. **提交到 GitHub**
   ```bash
   git add .
   git commit -m "新增文章：你的文章標題"
   git push origin main
   ```

5. **完成！** 1-2 分鐘後自動部署到網站

## 🎬 IG 短影音同步到網站

### **步驟：**

1. **在 IG 發布短影音**
   - 發布你的內容
   - 複製影片連結

2. **在部落格文章中嵌入**
   ```markdown
   ## 影片版本
   
   <div style="text-align: center; margin: 2rem 0;">
     <a href="你的IG影片連結" target="_blank" style="display: inline-block; background: linear-gradient(135deg, #833AB4, #FD1D1D, #F77737); color: white; padding: 1rem 2rem; border-radius: 12px; text-decoration: none; font-weight: bold;">
       📷 在 Instagram 觀看完整影片
     </a>
   </div>
   
   ## 文字版本
   
   （以下是影片的文字摘要）
   ```

3. **或直接嵌入影片（需要 IG 嵌入碼）**
   - 在 IG 影片點擊「...」→「嵌入」
   - 複製嵌入碼
   - 貼到文章中

## ✍️ 文章撰寫範本

### **補品營養類文章範本：**

```markdown
## 前言

我為什麼開始研究這個補品？（個人背景）

---

## 研究發現

根據研究文獻，這個補品的作用是...（引用來源）

---

## 我的實驗方法

- **劑量**：每天 XXX mg
- **時間**：持續 X 個月
- **追蹤指標**：血檢數據、主觀感受

---

## 數據變化

| 時間 | 指標 A | 指標 B | 感受 |
|------|--------|--------|------|
| 開始前 | XX | XX | ... |
| 3個月後 | XX | XX | ... |

---

## 關鍵心得

1. XXX
2. XXX
3. XXX

---

## 免責聲明

**這是我的個人實驗紀錄，不構成醫療建議。**

所有內容均基於個人經驗與公開研究資料分享，不應取代專業醫療人員的意見。
```

## 📋 文案用語檢查清單

### ✅ 可以使用：
- 「我的個人經驗」
- 「研究顯示」
- 「個人實驗紀錄」
- 「優化、調整、改善」
- 「僅供參考」

### ❌ 避免使用：
- 「診斷、治療」
- 「保證有效」
- 「可以治療 XX 疾病」
- 「替代藥物」

---

## 🎯 SEO 優化建議

### **標題優化：**
- 包含關鍵字（補品名稱、血檢項目等）
- 吸引點擊（數字、個人故事）
- 範例：「我如何用 Omega-3 降低 CRP 發炎指標：3個月實驗」

### **描述優化：**
- 1-2 句話說明文章重點
- 包含「個人經驗」框架
- 範例：「分享我使用 Omega-3 優化發炎指標的 3 個月實驗紀錄。這不是醫療建議，而是個人學習心得。」
