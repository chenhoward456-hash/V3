## 📝 製作流程

### 步驟 1：內容整理
- 從現有文章提取重點
- 整理成結構化內容
- 加入實用工具（檢查表、追蹤表）

### 步驟 2：設計排版
- 使用 Canva 或 Figma
- 簡潔專業的設計
- 加入品牌元素

### 步驟 3：加入 CTA
- 每頁底部放 LINE QR Code
- 「想要個人化指導？加 LINE 聊聊」
- 放網站連結

### 步驟 4：上傳和連結
- 上傳到網站 `/public/resources/`
- 在相關頁面加入下載連結
- 追蹤下載次數（GA）

## 📊 追蹤指標

### 每週檢查：
- 有多少人下載？
- 哪個資源最受歡迎？
- 下載後有多少人加 LINE？

### 優化方向：
- 如果下載率低 → 調整文案或位置
- 如果轉換率低 → 優化 PDF 內容和 CTA
- 如果某個特別受歡迎 → 做更多類似資源
