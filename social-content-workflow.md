## 工作流 SOP

### 流程概覽
```
部落格文章 → 拆解核心知識點 → 三種素材同步產出
                                  ├── Reels（15-45 秒）
                                  ├── 輪播圖文（8-10 頁）

```

### 每週發佈節奏
- **週一**：Reels 發佈（觸及新受眾）
- **週三**：限時動態 3 則（互動 + 預告）
- **週五**：輪播圖文發佈（深度內容留存）

### 製作流程（每篇約 30 分鐘）
1. 從文章抽出 1 個核心 hook + 3-4 個知識點
2. 寫 Reels 口播稿（hook → 痛點 → 知識點 → CTA）
3. 寫輪播每頁文案（封面 → 內容頁 → CTA 頁）
4. 寫限動素材（投票 → 知識卡 → 倒數/連結）
5. 套進 HTML 模板 or Canva 編輯
6. 排程發佈
